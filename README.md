# Domínio Completo do Terminal Linux

> Guia interativo e bilingue (PT/EN) do terminal Linux — 50 comandos explicados do básico ao avançado, num único ficheiro HTML autocontido.

![preview](social-preview.png)

## O que é

Um ebook interativo em HTML, CSS e JavaScript puro (sem frameworks, sem build step) que ensina o terminal Linux a partir do zero. Funciona como um "leitor" de página única: navegação por capítulos, índice lateral, barra de progresso e suporte total a PT/EN.

## Funcionalidades

- 🌐 **Bilingue** — troca instantânea entre Português e Inglês, incluindo os comentários dentro dos blocos de código
- 🌓 **Tema claro/escuro**
- 💾 **Retoma de leitura** — guarda localmente em que página, idioma e tema ficaste, e volta exactamente para lá da próxima vez que abrires o ficheiro
- 🔗 **Links directos por página** — `ebook.html#page-4` abre já naquela página
- 📦 **Ficheiro único e portátil** — `ebook.html` funciona offline, sem servidor, sem instalação; basta abrir no navegador
- ♿ Sem dependências externas em runtime além das Google Fonts (carregadas via CDN)

## Como usar

Só precisas do `ebook.html` — abre-o directamente no navegador (duplo clique ou `Ctrl+O`). Os ficheiros `ebook.css` e `ebook.js` têm de estar na mesma pasta.

```bash
git clone https://github.com/<teu-utilizador>/terminal-guide.git
cd terminal-guide
# abre ebook.html no navegador
```
